Android 4.x Quick Settings

2012

This is how I think the Android 4.2 quick settings pane should have looked. My version makes the toggle functionality much more apparent, displays information to the user while also putting text labels with the icons, moves the input method selector to quick settings (because it is a setting, not a notification), and allows the brightness setting to be adjusted directly in the panel.

The design in partially inspired by the Android 3.0 quick settings interface for tablets, the MotoBlur quick settings interface for Android 4.1 phones, and, obviously, the Android 4.2 quick settings interface.
