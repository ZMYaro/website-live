Google Wave Browser Extensions

2010-2011

Before Google Wave was shut down, I had developed or contributed to a few extensions and bookmarklets for it:

  • Wave Scroll Bars

    Extension & Bookmarklet

    Remove the custom scroll bars in the Google Wave client and restore the default browser scroll bars.

    This may stop working when Google Wave's layout gets updated, but I usually fix it within a few days at most.

    Extension
    In the Chrome Web Store     		Bookmarklet

  • Notifier for Google Wave

    Extension

    Displays your number of unread waves and notifies you of new ones.

    Extension
    In the Chrome Web Store     		Source code
    See it on GitHub

  • Pop-Out Wave

    Bookmarklet

    Opens the wave you are viewing in a new window. If you have multiple waves open it will only open the first one.

    Bookmarklet

  • Wave Reply

    Bookmarklet

    Click when reading a message in Gmail to turn the message into a wave where you can continue the conversation. Note that it will only take the first expanded message.

    Bookmarklet
    		Source code
    See it on GitHub
