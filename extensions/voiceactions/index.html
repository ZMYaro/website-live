Voice Actions for Chrome

Get for Chrome
Source Code Facebook Twitter

You talk, Chrome understands. Voice Actions for Chrome allows you to control your browser by talking to it. In addition to searching, it allows you to look up web pages, maps, directions, and music with spoken commands. It can also help you find a specific tab if you have too many open.
