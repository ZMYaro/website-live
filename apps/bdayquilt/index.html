Birthday Quilt Maker

2015 Open Quilt Maker
Source Code

This simple app takes a list of birthdays and turns them into colors using one of several algorithms. The birthday colors can then be exported as a “quilt” of colored squares. The app is optimized for use on mobile devices and can be pinned to a smartphone's home screen and launched as its own activity.
