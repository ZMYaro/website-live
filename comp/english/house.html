The House (Organizer)



This is the house graphic organizer. It is used to organize your thoughts becore you write an essay, short story, etc.

There are two links below. The .doc file is a Microsoft Word Document and the .dot file is a Microsoft Word Template.

house_organizer.doc
Right-click on this link and save it wherever you want.

house_organizer.dot
Save this file to C:\Documents%20and%20Settings\[your user name]\ApplicationData\Microsoft\Templates or C:\Documents and Settings\[your user name]\ApplicationData\Microsoft\Templates. Note: these are instructrions for Windows XP.
