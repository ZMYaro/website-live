What is a "Normal.dot"?

Open Microsoft Word. Notice the toolbars, zoom level, page view, and other settings. These are all (for the most part) saved in a special document template called "Normal.dot". One of the things that makes Normal.dot different from other document templates (other than that is the default template) is that it contains the toolbars. If you want someone's custom toolbar, you need that person's Normal.dot. My Normal.dot contains a toolbar and keyboard shortcuts for many math symbols.
