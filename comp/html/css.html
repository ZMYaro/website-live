What is CSS and how is it better?

    CSS is a better way to format text and is an easy alternative to the <font>, <b>, <i>, and <u> tags. It can be created in a stylesheet (a Cascading Style Sheet), put in the. <head> using the <style> tag, or by using the style attribute in a tag. The last one is usually for short bits of CSS, not whole classes (that is what stylesheets are for).
