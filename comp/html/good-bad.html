Good vs. Bad in HTML and web page design

Good Bad
  • Use CSS!
  • Use non-depricated code!
  • Using the <font> tag AT ALL.
  • Using too many <hr>s (horizontal rules).
  • Do not use bold, italics, and underline at the same time. There are better and more creative ways to make a point.
  • Using the <marquee> tag. It is not depricated, but it is annoying to most users. Only use it occasionally.
  • Using the <center> unless necessary (I do not mean do not center text or images; I mean do it with a non-depricated tag or CSS).
  • Using huge images (such as EMFs) that take too long to load.
