Creating a table

A table starts with the <table> tag (easy enough to remember, right). Inside that tag, you must create a row using the <td> tag (get it? table row). Then you can create cells. If the cell is a normal data cell, you use the <td> (just like a row, table data). You can also reate a header cell, which automatically centers and bolds the text, using the <th> tag (like table header/table heading).
