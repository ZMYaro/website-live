For those who need to learn HTML...or at least proper HTML


What is HTML?

HTML is the programming language used to create webpages (for those who do not know, a webpage is a page on the Internet like the one you are looking at right now). HTML is a rather free language. Unlike most programming languages, it does not require specific pieces of code to be absolutely flawless and it allows the programmer to choose ther order of attributes, when to use CaPs, etc. Let us (or rather, you) get started.


The basics

HTML is divided into two sections: the <head> and the <body> (Note: "<" and ">" signs, referred to, in HTML, as "angle brackets," identify tags. Tags are used for creating all parts of a webpage, such as hyperlinks, the title bar text, colors, fonts and sizes, and images). All code is inside the <html> tag, which tells the browser that it is about to read HTML code. For the most part, things that will affect and run the page are found in the <head> and things that actually appear on the page are found in the <body>.

This is the most basic (proper) webpage you can create:

<html>
      <head>
      </head>
      <body>
      </body>
</html>

"What is with the indenting?", you ask? Well I will tell you. If there is only one piece of advice I give you that you remember, remember this: SPACE YOUR CODE!. Got it? Also, about CaPS, you could create your <html> tag in any of these ways: <HTML>, <hTmL>, <HtMl>, <HtmL>, <hTMl>, <hTML> <htML>, or any other combination of CAPS and lowercase you can think of and still get the same result, but I reccomend all lowercase.
