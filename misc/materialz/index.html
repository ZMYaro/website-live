MaterialZ

2014-present MaterialZ Website
Source Code

MaterialZ is a pure CSS implementation of Material Design. Simply adding the basic MaterialZ style sheet will theme HTML elements on a page, and additional style sheets can be added to enable dark theme, depth shadows, and widgets such as cards, switches, progress spinners, and app bars.

MaterialZ supports as many elements as possible without additional code, including newer HTML5 elements such as <input type="range" /> and <progress>. The goal is for you to be able to drop MaterialZ into an unstyled page and have it “just work”.
