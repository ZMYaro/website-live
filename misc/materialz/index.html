MaterialZ

MaterialZ Website
Source Code

MaterialZ is a pure CSS implementation of Material Design. Simply adding the basic MaterialZ stylesheet will theme HTML elements on a page, and additional stylesheets can be added to enable depth shadows and widgets such as cards and app bars.

MaterialZ supports as many elements as possible without additional code, including newer HTML5 elements such as <input type="range" /> and <progress>.
