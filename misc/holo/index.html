Holo Web

2012-2014 Holo Web Website
Source Code

Holo Web is a few stylesheets that imitate the Android “Holo” themes. Simply adding the stylesheets will instantly Holo-theme any site. With a few tweaks, the site can also add more Android-like UI elements, such as Holo-themed action bars.

Holo Web supports as many elements as possible without additional code, including newer HTML5 elements such as <input type="range" /> and <progress>.
